# logi-OS — Project Brief

## Kernziel
Operations System / Dashboard für die Logistik-Abteilung von Orendt Studios.

## Scope
- Interne Web-App für das Logistik-Team
- Schrittweise Feature-Erweiterung
- MVP: Abwesenheitsplaner (Urlaubsübersicht)

## Tech-Stack
- **Frontend:** React 19 + Next.js 16 App Router
- **Backend:** Supabase (PostgreSQL, Auth, Storage, Realtime)
- **Styling:** Tailwind CSS v4 nach Orendt Brand Guideline
- **Icons:** Lucide React
- **Fonts:** Sora (Display) + Instrument Sans (Body)
- **Deployment:** Vercel (geplant)

## Zielgruppe
- Interne Mitarbeiter von Orendt Studios (Logistik + Produktion)
- Duzen, kollegialer Ton, deutsche UI
