# Product Context

## Problem
Das Logistik-Team von Orendt Studios braucht eine zentrale Übersicht, wer wann abwesend ist. Aktuell gibt es keine einfache Möglichkeit zu sehen, wer Urlaub hat, krank ist oder im Homeoffice arbeitet.

## Lösung
logi-OS bietet eine Kalender-Übersicht (Mitarbeiter × Tage) mit farbkodierten Abwesenheitstypen. Jeder Mitarbeiter kann eigene Abwesenheiten eintragen.

## Abwesenheitstypen
| Typ | Farbe | Use Case |
|-----|-------|----------|
| Urlaub | Gelb (Accent) | Geplanter Urlaub |
| Frei | Blau | Freier Tag (Gleittag etc.) |
| Früher gehen | Orange | Vorzeitig den Tag beenden |
| Krank | Rot | Krankheit |
| Homeoffice | Grün | Von zuhause arbeiten |

## User Experience
- **Schnell**: Kalender sofort sichtbar, keine Klicks nötig
- **Klar**: Farben zeigen sofort den Status
- **Einfach**: Eintragen per Modal, wenige Felder
- **Team-Fokus**: Alle sehen alle — Transparenz im Team
